| 字段 | 内容 |

### 变更清单

- **新增（M1）**：生活流表结构与迁移；DeepSeek 客户端；周大纲/日场景/她的宇宙/文案/生图/每日发布流水线；Feed 列表/点赞/评论/评论回复；`feed.html` 与首页入口双徽标。

- **新增（M3）**：生活流后台 API + 8 管理页 + 菜单 RBAC（含 ops 只读）。

- **文档（2026-07-12）**：M1+M2+M3 契约草案合并进 `docs/contract.md`；草案目录保留快照。

### 运维备注

- 迁移：`v6a` / `v6b` / `v6c` / `v6d`（`reply_to_lxm`）/ `v6e`（`base_comments` / `comment_multiplier`）。

- 环境变量：`DEEPSEEK_*`、`LIBLIB_*` 等（见 `.env.example` 与 `contract.md` 生活流节）。
- **正式发布前**：打 `git tag v2.0.0`（契约合并已完成）。

- IM 主链 v1 不注入 Feed / 世界观内容。

### 备注

## 维护约定

1. **何时追加版本**：完成一次可独立验收 / 可对外说明的功能包，或需要独立回滚点时。
2. **何时只改修订号**：纯修 bug、超时/并发等运维补丁、无新能力面。
3. **文档同步**：有接口/库表变更仍须更新 `contract.md`（或阶段草案）；本表只写概述，不复制字段级契约。
4. **进度文档**：STEP 勾选仍在 `docs/progress/`；本表用「关联」链过去即可。
5. **tag**：`v2.0.0` 正式合并契约并入库后，建议执行 `git tag v2.0.0`。
