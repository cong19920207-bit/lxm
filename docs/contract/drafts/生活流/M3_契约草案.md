## 一、后台管理 API

> 统一约定：前缀 `/api/admin`；响应格式 `{code,data,message}`；除标注外均需管理员 JWT（有效期 2h，与用户端独立）；写操作落 `operation_log`；配置类写操作走 admin_config 草稿三卡点（草稿→测试集→CONFIRM→5min 监控）。RBAC 见「四、RBAC」。

### 1.1 生活计划（`life_plan_mgmt` · STEP-006/008）

| 方法 | 路径 | 权限 | 说明 |

| PUT | `/life-plan/settings` | 编辑角色 | 写上述设置（home_city 非空校验；**仅存草稿**，不立即生效；发布走 `/life-config/publish`） |

| PUT | `/life-plan/daily/{plan_date}/scenes/{scene_id}` | 编辑角色 | 编辑场景；**与新增一致：category 必须 ∈ 当日大纲**（2026-07-09 定案 B）；city 不强制 = 大纲城市；不主动降级 gen_status |
| DELETE | `/life-plan/daily/{plan_date}/scenes/{scene_id}` | 编辑角色 | 删除场景；scenes&lt;2 时不主动降级 gen_status |

| 入参 | 返回 | 排序 |
